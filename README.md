# Intro

인공지능을 활용하여 시스템의 취약점을 선제적으로 탐지하고 분석하는 퍼징(Fuzzing) 프로젝트 그룹입니다.

## Tech Stack & Environment

- **Language:** Python, C/C++ and ETC
- **Virtualization & Container:** Docker, QEMU
- **Infrastructure:** High-throughput Fuzzing Cluster (Optimized for Multi-core & Large RAM Environments)

## Core Objectives

1. **AI-Guided Mutation:** 기계학습 모델을 활용한 스마트 뮤테이션 전략 연구 및 적용
2. **Scalable Architecture:** 컨테이너 기반의 병렬 처리를 통한 퍼징 커버리지 극대화
3. **Cross-Architecture Analysis:** QEMU 에뮬레이션을 통한 다양한 타겟 아키텍처 취약점 분석
